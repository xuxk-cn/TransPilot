# TransPilot

`TransPilot` 是一个面向 Windows 的截图、识图、翻译与表格识别应用。

该项目当前以闭源软件形式发布，GitHub 仓库主要用于：

- 产品介绍
- 发布下载说明
- 使用文档
- 部署文档
- 版本更新记录

GitHub 仓库：
[https://github.com/xuxk-cn/TransPilot](https://github.com/xuxk-cn/TransPilot)

## 产品定位

`TransPilot` 主要面向以下两类用户：

### 1. B2B / 团队内部工具

适用场景：

- 外贸
- 制造业
- 海外资料阅读
- 招投标 / 预算表识别
- 内网环境
- 数据敏感部门

核心价值：

- 截图与图像内容可以本地处理
- 不需要把业务图片上传到外部云服务
- 可以统一接入公司服务器上的 `llama.cpp`
- 适合团队共享、集中维护

### 2. 专业个人用户

适用人群：

- 需要长期处理外语截图
- 不信任云端上传
- 希望本地模型可控

核心价值：

- 本地部署
- 使用路径清晰
- 更适合高频办公与资料处理

## 核心功能

- 截图-识图-翻译
- 表格识别并导出 Excel
- 保存截图图片
- 支持本地 `llama.cpp + Qwen2.5-VL`
- 支持接入公司内部部署的 `llama.cpp`
- 支持界面语言与翻译目标语言切换

## 运行方式

### 方式 1：完整包

适合普通用户。

完整包已包含：

- `TransPilot.exe`
- `llama.cpp`
- `Qwen2.5-VL-7B-Instruct-q4_k_m.gguf`
- `mmproj-F16.gguf`
- 默认配置文件

用户解压后即可运行。

### 方式 2：标准包

适合：

- 已有本地模型环境的用户
- 已有公司服务器模型服务的用户

用户只需配置服务地址即可接入。
