# Changelog

## v1.0.0 - 2026-04-11

正式 `1.0.0` 版本。

### 新增

- Windows WPF 主程序
- 截图-识图-翻译
- 表格识别并导出 Excel
- 保存截图图片
- 本地 `llama.cpp + Qwen2.5-VL` 支持
- 公司服务器 / 内网部署接入能力
- 多语言设置入口
- 应用图标与主界面 Logo
- 帮助文档
- 普通用户下载版帮助文档
- 管理员部署版帮助文档

### 调整

- 优化启动阶段流程，窗口优先显示，服务检查改为后台快速执行
- 调整主界面布局与按钮区域
- 完整包支持解压后直接运行

### 发布内容

- 标准发布包：`TransPilot-v1.0.0.zip`
- 完整发布包：`TransPilot-v1.0.0-full.zip`
